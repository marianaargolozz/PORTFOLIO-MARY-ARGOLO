MARY ARGOLO — SITE PORTFÓLIO

Atualizações desta versão:
- Seção "Antes e Depois" removida.
- Destaque alterado para +3 anos.
- Badge alterado para Desde 2021.
- Capas dos vídeos refeitas em estilo minimalista e limpo.
- Serviços principais com cores uniformes e ícones em linha roxa/lilás.
- Portfólio continua por categoria: a pessoa escolhe a seção e só depois os vídeos aparecem.
- Layout responsivo para celular.

Como testar:
1. Extraia o ZIP.
2. Abra index.html no navegador.
3. Clique em uma categoria no portfólio para abrir os vídeos.

Como publicar no GitHub Pages:
1. Crie um repositório no GitHub.
2. Envie index.html, style.css, script.js e a pasta assets.
3. Em Settings > Pages, publique pela branch main.
