MARY ARGOLO — PORTFÓLIO PROFISSIONAL

Esta versão foi atualizada com:
- nova foto principal no início do site;
- ícones padronizados e coerentes com cada formato;
- cores uniformes nos cards de serviços e categorias;
- portfólio por categorias: os vídeos só aparecem após clicar no formato;
- animações suaves e profissionais nos cards, seleção de categoria, modal e elementos da página;
- responsividade para celular;
- aviso do Google Drive removido.

Para testar:
1. Extraia o ZIP.
2. Abra index.html no navegador.
3. Clique em uma categoria do portfólio para ver os vídeos.
4. Clique em um vídeo para abrir o modal e assistir.

Para publicar no GitHub Pages:
- Suba index.html, style.css, script.js e a pasta assets para o repositório.
- Ative GitHub Pages nas configurações do repositório.
